# HUD WebGL Checklist

Проверять в landscape, после загрузки игры и после 1-2 ресайзов окна.

## Desktop

- `1920x1080` (16:9)
- `1600x900` (16:9)
- `1366x768` (16:9 low height)
- `1280x720` (baseline)
- `1728x1117` или любое окно около `3:2`
- `2560x1080` (21:9 ultrawide)

Проверить:
- верхний HUD не прилипает к краям и не обрезается;
- главное меню помещается целиком;
- `Shop`, `Settings`, `Game Over` не наслаиваются кнопками на текст;
- после изменения размера окна HUD пересобирается без смещения.

## Mobile WebGL

- `932x430` CSS px, iPhone 14 Pro Max landscape
- `852x393` CSS px, iPhone 14 / 15 landscape
- `812x375` CSS px, iPhone X / 11 Pro landscape
- `915x412` CSS px, Android 20:9 landscape
- `800x360` CSS px, Android compact landscape
- `1024x600` CSS px, small tablet landscape

Проверить:
- HUD не уходит под вырез / dynamic island / системные зоны;
- карточки `Brand`, `Score`, `Distance`, `RAM`, `Coins` полностью видны;
- нижняя подсказка управления не упирается в жестовую область;
- меню центрируется внутри safe area;
- на низких экранах нижний блок формулы в главном меню либо виден целиком, либо корректно скрыт без наложения на кнопки.

## Edge Cases

- открыть игру на телефоне, свернуть и вернуть;
- повернуть устройство portrait -> landscape, если платформа это допускает;
- открыть рекламу / overlay платформы и вернуться;
- на desktop поменять масштаб браузера `80%`, `100%`, `125%`.
